# 情感倾向性分析

疫情期间微博内容的情感倾向性分析  
3种类别，负向（-1）、中性（0）、正向（1）

## CoarseLabel_converge.ipynb  
使用1-tag，4个模型  
1. Logistic Regression(LR)  
2. Support Vector Machine(SVM)  
3. MultiLayer Perceptron(MLP)  
4. Random Forest(RF)  

## CoarseLabelLR.ipynb  
使用Logistic Regression(LR)，3种特征提取  
1. 1-tag
2. 2-tag
3. 1+2-tag
